# رفع المشروع إلى GitHub — خطوة واحدة

المشروع جاهز للرفع. تحتاج **مرة واحدة** أن يعرف Git حسابك عبر **توكن** (Token).

---

## الخطوة الوحيدة: إنشاء توكن ثم رفع

### 1) إنشاء التوكن (دقيقة واحدة)

1. افتح هذا الرابط (سجّل الدخول إن طُلب منك):  
   **https://github.com/settings/tokens/new**
2. **Note:** اكتب مثلاً: `khawam`
3. **Expiration:** اختر مدة (مثلاً 90 يوم).
4. تحقق من **repo** (كل الصلاحيات تحته).
5. اضغط **Generate token**.
6. **انسخ التوكن فوراً** (يظهر مرة واحدة فقط؛ احفظه في مكان آمن).

---

### 2) تنفيذ الرفع

افتح **PowerShell** أو **Terminal** في مجلد المشروع ثم نفّذ:

```powershell
cd "c:\Users\waeln\Documents\GitHub\khawam"
git push -u origin main
```

عند الطلب:

- **Username:** `wael7705`
- **Password:** الصق **التوكن** الذي نسخته (وليس كلمة مرور حسابك في الموقع).

بعدها يتم الرفع وتصبح كل الملفات على: **https://github.com/wael7705/khawam**

---

## إذا لم يطلب منك اسم مستخدم وكلمة مرور

قد تظهر نافذة تسجيل دخول من Windows أو من المتصفح — سجّل الدخول بحساب **wael7705** من هناك، ثم أعد تشغيل:

```powershell

```
